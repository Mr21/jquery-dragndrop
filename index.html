What is it ?

This plugin offers a fluide drag n drop with great visual feedback.
jquery-dragndrop can works alone but if you want the possibility to select several items in the same time with the Ctrl/âŒ˜ and Shift keys, you have to include also this one : jquery-selection.

This two plugins work great together !

Needs

This plugin need 3 or 5 files :

  • jquery.js (obviously)
  • jquery-dragndrop.js
  • jquery-dragndrop.css
  • jquery-selection.js (optional)
  • jquery-selection.css (optional)
